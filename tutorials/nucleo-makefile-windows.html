Instalace vývojových nástrojů pro Nucleo na Windows

Postupujte podle následujících kroků:

Instalace mingw

Prvně, abyste na windows mohli spustit makefile - což se soubor instrukcí pro kompilaci, popř. instalaci - potřebujete MinGW. Přečtěte si pozorně instrukce v úvodu k práci s MinGW zde: http://www.mingw.org/wiki/Getting_Started Podle tohoto návodu MinGW nainstalujte.

Instalace kompieru a debuggeru

  1. Na následující stránce si stáhněte instalační program nástrojů pro vývoj: https://launchpad.net/gcc-arm-embedded/+download Jedná se o kompiler, linker a debuger.
  2. Instalátor spusťte a nástroje nainstalujte.
  3. Poznámka: nezapomenout přesunout ukázkový projekt na CWNyní je čas na stažení ukázkového projektu. Následující krok se liší, pokud nemáte GIT.
    1. Pokud máte GIT: ve vhodné složce si stáhněte repozitář tímto přikazem: git clone git@github.com:Darker/PRG_nucleo_template.git
    2. Pokud nemáte GIT: stáhněte si archiv s projektem jako ZIP: https://github.com/Darker/PRG_nucleo_template/archive/master.zip

Kompilace:

  1. Připojte Nucleo k počítači. Mělo by se zobrazit jako vyměnitelné médium pojmenované NODE_F446RE.
  2. Ukázkový projekt nyní zkompilujte příkazem mingw32-make. Pozorně sledujte výstup.
